MarketBook — Landing Page

Marketing landing page for MarketBook, a profit-tracking app built for 
African traders and shop owners.

Built with vanilla HTML, CSS, and JavaScript. No frameworks, no build step.
Deploy directly to GitHub Pages.

- Pure HTML/CSS/JS
- Cormorant Garamond + Instrument Sans (Google Fonts)
- Scroll reveal animations
- Responsive (mobile + desktop)
